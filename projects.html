Spoon Me

This is an application that lets people search for recipes. Once logged in the user can save recipes, view their saved recipes, edit them, add notes to make them their own. They will also have the ability to create original recipes.

HTML, CSS, JavaScript, PostgreSQL, Express, React.js, Node.js, Redux, Saga, Axios, Passport, Material-ui, Spoonacular API, Sweet Alerts

Todo App

An app where you can add, complete and delete tasks. Manage your tasks by adding a due date. after the due date has passed the task will be highlighted in red. When the task is completed it will be highlighted in yellow. The delete button will first confirm you mean to delete the task and then remove the task from your list permanently.

HTML, CSS, JavaScript, PostgreSQL, Express, React.js, Node.js, Axios, Sweet Alerts

Server-side Calculator App

A calculator that has all the basic operators. As well as displays the calculation history, which can be cleared with the AC button. All logic for this calculator was done on the server-side.

HTML, CSS, JavaScript, jQuery, Express, Node.js, Axios
